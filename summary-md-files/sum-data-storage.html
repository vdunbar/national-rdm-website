Section Summary

  • When storing data, it’s worth considering seperate storage platforms for data you are actively working with, as well as long-term storage for files that aren’t frequently used or that are part of a project that has been completed.

  • Developing a basic understanding of the differences between hard drives, network storage, and cloud storage is very helpful knowledge is selecting how you store data.

  • Backing up your data, ideally with the 3-2-1 model in which data is stored in multiple physical locations and backups are done routinely, is a solid solution to preventing data loss.

  • It’s always good to be conscious of how much storage space you have, and if you think you might reach the limit, arrange for additional storage in advance to avoid project disruptions.

  • If your data has any privacy or security restrictions, ensure that your selected storage platforms are suitable.
